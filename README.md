## 🛠️ 学习必备工具清单（持续更新）
### 1. 信息收集类
- Nmap：端口扫描、服务探测
- Subfinder：子域名收集
- Ffuf：目录/接口爆破
- Dnsx：域名解析探测
- 钟馗之眼：网络空间测绘

### 2. 漏洞扫描类
- Nuclei：自动化漏洞扫描、POC批量检测
- Xray：主动/被动漏洞扫描
- Afrog：轻量化漏洞扫描器

### 3. Web渗透类
- Burp Suite：抓包改包、漏洞利用（渗透测试核心工具）
- Sqlmap：SQL注入自动化利用
- AntSword蚁剑：Webshell管理工具
- BeEF：XSS漏洞利用框架

### 4. 内网渗透类
- Metasploit(MSF)：漏洞利用框架
- Mimikatz：Windows密码抓取
- CrackMapExec：内网横向批量探测
- Cobalt Strike：内网渗透协同框架
